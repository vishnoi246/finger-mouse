Finger Mouse

A simple Python project that lets you control your mouse cursor using hand gestures. It tracks your finger in real-time and moves the cursor when you raise one finger.

Features:
Real-time hand and finger detection
Moves mouse cursor based on finger position
Uses OpenCV and MediaPipe for hand tracking
Controls cursor with PyAutoGUI

Requirements:
Python 3.x
OpenCV
MediaPipe
PyAutoGUI
NumPy
